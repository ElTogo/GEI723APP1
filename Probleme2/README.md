Le code est divisé en deux fichiers, le fichier "partie1.py" est la partie I du problème tandis que le fichier "neuro_2.ipynb" est les parties II à IV

Pour le fichier "partie1.py", le code est adapté de l'exemple de STDP de brian2 se trouvant sur le lien suivant: https://brian2.readthedocs.io/en/latest/examples/synapses.STDP.html

Ce code est divisé en quatre section, soit aI, aII, bI, bII. pour lancer les graphiques des différentes STDP, il faut changer les trois variables se trouvant à la ligne 79, soit eqs_x, on_pre_x et on_post_x.

Pour la partie 2.b, la fréquence peut être changée à la ligne 11

Pour le fichier "neuro_2.ipynb", le notebook jupyter est un code à compléter avec des valeurs à modifier. Ces valeurs ont étés choisies à partir de l'exemple original de Diehl et Cook https://www.frontiersin.org/articles/10.3389/fncom.2015.00099/full

Pour exécuter le code, simplement cliquer sur "tout exécuter" et les blocs seront exécutés en ordre
